J-Lift

只记录，不教动作。
给我自己做的运动工具，刚好也适合你这样的人。

为什么有它

我不满意所有运动 app。它们要么想卖我课，要么 UI 太吵，要么不让我语音记组数。我用心率带配套 app 记录健身和骑行，用华为健康管日常消耗和生理期，还想随时记一下腰围——但不想为这些需求装四个 app。

AI 时代了，奢侈一下：给自己写一个。

它能做什么

🎙 语音记组 撸铁时对手机说"卧推 40kg 10次 第3组"，自动记录。不用碰屏幕。
📊 心率联动 接入心率带数据，看每个动作整组的用时与心率变化曲线。
📏 体围追踪 腰围为主。每周量一次，画条线就够了。不搞体脂率焦虑。
🩸 日常 & 生理期 华为健康数据的轻量看板，不替代它，但不用切 app 看。

适合谁

  • → 稳定撸铁 2 年以上的习惯型训练者
  • → 只想记个数，不打算学动作
  • → 不玩跟练、不买课
  • → 身材保持理想，定期量腰围就行
